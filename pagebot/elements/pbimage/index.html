Image

The Image element is a “normal” container, which contains one (or more) PixelMap elements and zero (or more) caption or other elements. This way the user can add mulitple PixelMaps, a title elements, etc. The layout of the Image elements is defined in the same way as any other layout. Conditional rules can be applied (e.g. if the image element changes size), or the child elements can be put on fixed positions.
